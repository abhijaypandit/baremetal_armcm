# Bare metal ARM Cortex-M3
This repo contains bare metal code examples for ARM Cortex-M3 processor. These examples target the **Stellaris
LM3S811 microcontroller**.

## Prerequisites
To get started, make sure the following tools are installed on your system. The instructions below are specific to macOS using homebrew package manager.

1. QEMU
`brew install qemu`

2. ARM GNU toolchain
`brew install arm-none-eabi-gcc`

3. ARM GNU debugger
`brew install arm-none-eabi-gdb`
